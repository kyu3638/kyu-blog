### # 개요

- 고객 현장 설비를 제어하는 IoT 장비에 대한 모니터링 및 설정을 수정하는 백오피스 페이지

### # Skills

- React, Typescript, jotai, react-query, emotion, reach-hook-form

### # 역할

#### Frontend

- 백오피스 페이지 마이그레이션
  - ASP.NET Razor Pages -> React
  - API 응답 인터페이스를 정의하여 DX 향상
- 최적화
  - 가상화를 통해 최대 9000여개 데이터 렌더링 성능을 향상
- storybook 도입으로 개발 및 유지보수 비용 감소

#### Backend

- ASP.NET Razor Pages를 Restful API로 마이그레이션
- Entity 별 DB와 Cache 동기화 여부를 확인하고 갱신
